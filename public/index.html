JS Objects

Creating Objects in Javascript:

  • var obj1 = new Object();
  • var obj2 = {};
  • var obj3 = Object.Create(Object);

JS Objects

Acessing Object properties:


        // dot notation
        obj.name = 'MyObject';        
        obj.name; //'MyObject'

        // bracket notation
        obj['name'] = 'MyObject';
        obj['name']; //'MyObject'

JS Objects

Methods:


        obj.print = function () {
          console.log(this.name);
        };
        obj.print(); //'MyObject'

JS Objects

Object literal:


        var person1 = { 
          name:"John" ,
          greet: function() { 
            console.log("Hi I'm" + name);
          }
        };

JS Objects

Constructor:


        var person2 = new Object();
        person2.name = "John";
        person2.greet = function() { 
            console.log("Hi I'm" + name);
          };

JS Objects

Object.Create()


        var person3 = Object.Create(Object);
        person3["name"] = "John";
        person3["greet"] = function() { 
            console.log("Hi I'm" + name);
          };

JS Objects

Javascrips Objects are peculiar.

But very flexible!

  • They are similar to Hash Maps
  • Methods and Properties are created in the same way
  • you can add properties / methods at any time
  • they don't have classes (yet - coming in ES6)
    • but they have prototypes!
