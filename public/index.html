Acessing Object properties:


        // dot notation
        obj.name = 'MyObject';        
        obj.name; //'MyObject'

        // bracket notation
        obj['name'] = 'MyObject';
        obj['name']; //'MyObject'

JS Objects

Methods:


        obj.print = function () {
          console.log(this.name);
        };
        obj.print(); //'MyObject'

JS Objects

Object.create( )

With Object.create( ) we use an object as the prototype for other objects.

This new object will inherit its properties from the prototype object.

New properties/methods added to the prototype object and they will be available in all created objects.
